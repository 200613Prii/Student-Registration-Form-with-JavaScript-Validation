# Student Registration Form with JavaScript Validation

## 📌 Project Overview
This project is a simple Student Registration Form built using HTML, CSS, and JavaScript. It allows users to enter their details and validates the input fields before submission.

## 🚀 Features
- User-friendly and clean interface
- Form validation using JavaScript
- Validates:
  - Name (letters only)
  - Email format
  - Password length (minimum 6 characters)
  - Gender selection
  - Course selection
  - At least one hobby selection
  - Address field (cannot be empty)
- Reset button to clear form data

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
